Task Management Chatbot:

This project is a Python-based command-line chatbot that helps users manage daily tasks. Tasks are stored locally using a JSON file to ensure data persistence between program runs.

Features:

     Add new tasks 
  
     View existing tasks
  
     Persistent storage using JSON

Technologies Used:

     Python 3
  
     JSON
  
     datetime module

Project Structure:

     projectbot.py   - Main chatbot logic
   
     tasks.json      - Task storage file
  
     .gitignore      - Git ignored files
  
     README.md       - Project documentation

How to Run:

     Ensure Python 3 is installed
  
     Open terminal in the project directory
  
     Run the following command:
                python projectbot.py

Limitations:

     Command-line only
  
     No natural language processing
